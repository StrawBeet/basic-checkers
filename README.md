# Basic Checkers
Game of checkers where you can play against a friend via local multiplayer or against a bot

Note that this game will only run on Windows

To play the game you simply run the executable once you have unzipped the Checkers.zip file that you download from releases

After opening the game, simply choose the gamemode you wish to play from the main menu
You can also change the color of the black squares to a different one if you wish, although there's a limited amount of options

The game follows the standard Checkers rules:
Captures are mandatory, you may capture multiple pieces in a row if you wish to, and your turn ends immediately after you promote a piece
The game also attempts to estimate how well you play and gives you a rating from 1 to 3 points after the game ends
